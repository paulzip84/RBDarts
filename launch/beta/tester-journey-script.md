# Beta Tester Journey Script

Status: Draft.

## Script

1. Install the beta candidate from the platform testing channel.
2. Confirm the app name is RBDarts and no development labels are visible.
3. Sign in with the assigned SSO provider.
4. Create a standalone baseball darts game with two sample players.
5. Enter valid scores across multiple innings.
6. Try one invalid score and confirm it is rejected without losing data.
7. Background the app, return, and confirm scoring state remains intact.
8. Restart the app and confirm active scoring recovery.
9. Complete the game and review the final summary.
10. Open privacy, support, and account deletion links.
11. Sign out.
12. Submit feedback with device model, OS version, app build, and screenshots when useful.

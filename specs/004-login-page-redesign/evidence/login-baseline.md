# Login Baseline And Design Delta

Feature: `004-login-page-redesign`

## Baseline

Before this feature, the Android login screen used a simple light Material layout with:

- RBDarts text heading.
- `Secure SSO` label.
- Google and Facebook actions.
- Full privacy/support/account deletion block displayed directly on the page.

## Design Delta

The redesigned implementation introduces:

- Dark premium full-screen login surface.
- RBDarts mark above the welcome heading.
- `Welcome back` headline.
- Concise SSO-only supporting copy.
- Provider-aware Google/Facebook buttons with loading and disabled states.
- Compact secondary support links.
- Explicit no-password/no-email/no-reset guardrails.
